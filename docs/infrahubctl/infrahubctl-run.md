# `infrahubctl run`

Execute a script.

**Usage**:

```console
$ infrahubctl run [OPTIONS] SCRIPT
```

**Arguments**:

* `SCRIPT`: [required]

**Options**:

* `--method TEXT`: [default: run]
* `--debug / --no-debug`: [default: no-debug]
* `--config-file TEXT`: [env var: INFRAHUBCTL_CONFIG; default: infrahubctl.toml]
* `--help`: Show this message and exit.
